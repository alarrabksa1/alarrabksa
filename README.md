# مشروع أبواب العراب 🚪

موقع إلكتروني لمصنع أبواب العراب - متخصصون في صناعة الأبواب والأنظمة الأوتوماتيكية

## 🌟 المميزات

- تصميم عصري وجذاب
- واجهة مستخدم سهلة الاستخدام
- متجاوب مع جميع الأجهزة
- معرض صور احترافي
- نظام تواصل متكامل
- قسم خاص للمنتجات والخدمات

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Bootstrap 5
- Swiper.js

## 📂 هيكل المشروع

```
├── html/
│   ├── صفحات المنتجات/
│   ├── خدمات الموقع/
│   └── photo/
├── style/
│   └── style.css
├── javascript/
│   └── main.js
└── main.html
```

## 🚀 كيفية التشغيل

1. قم بتنزيل الملفات أو نسخ المستودع
2. افتح ملف `main.html` في المتصفح
3. استمتع بتصفح الموقع!

## 📱 التوافق

الموقع متوافق مع:

- جميع المتصفحات الحديثة
- الأجهزة المحمولة
- الأجهزة اللوحية
- أجهزة سطح المكتب
